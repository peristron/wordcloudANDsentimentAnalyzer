

    Multi-File Word Cloud & Sentiment Analysis Generator

    This is a high-performance Streamlit tool designed for quickly analyzing text from large local files. It serves as an interactive dashboard for anyone needing a "bird's-eye view" of textual data without the overhead of a database.

    Key Features:

        Multi-File Upload: Process multiple CSV, Excel (.xlsx/.xlsm), and WebVTT (.vtt) transcript files at once.
        Word Cloud Generation: Instantly creates customizable word clouds for both individual files and the combined text corpus.
        Sentiment Analysis: Leverages NLTK's VADER to analyze and color-code words, providing an at-a-glance understanding of the text's emotional tone.
        Advanced Cleaning & Filtering: Includes powerful options to remove common chat/transcript artifacts, HTML tags, URLs, and custom stopwords to refine your analysis.
        High-Performance Design: Built with a streaming, memory-safe architecture to handle files of several hundred megabytes directly in the browser on public servers.
        Downloadable Outputs: Easily download the generated word cloud images (PNG) and frequency tables (CSV) for further use.
